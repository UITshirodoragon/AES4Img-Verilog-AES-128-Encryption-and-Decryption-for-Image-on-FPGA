AES4Img

Verilog AES-128 image encryption/decryption demo for 320x240 RGB565 images, streamed from PC to FPGA and visualized live on VGA.

Live image crypto pipeline

One board, three memory views, four VGA quadrants.

UART Image Load

PC sends 600 validated packets into external SRAM before AES starts.

AES DMA

DMA packs 8 RGB565 pixels into each 128-bit AES block and writes the result back.

VGA Dashboard

Original, encrypted, decrypted and status views appear together at 640x480.

Run path

Build the FPGA image, stream pixels, press start.

Quartus

  1. Open quartus/AES_128_v2_1_uart.qpf.
  2. Confirm top-level entity is top_uart.
  3. Compile and program the generated FPGA bitstream.

Python Sender

python -m pip install pillow pyserial tkinterdnd2
python tools/send_image_packet_2.py --gui
python tools/send_image_packet_2.py --path tools/test0.png --port COM3

Controls

SW0FAST / SLOW-L3
